# HR Management System

## 概要
Java + Spring Boot を用いた社員管理システム

## 機能
- ログイン機能（Spring Security）
- 社員CRUD
- 検索機能
- ユーザー管理
- REST API
- バリデーション
- エラーハンドリング

## 技術スタック
- Java
- Spring Boot
- MySQL
- Thymeleaf
- JavaScript（fetch API）

## API例

GET /api/employees

POST /api/employees

## 特徴
- DTOによるデータ分離
- 共通レスポンス設計
- グローバル例外処理
